Hva er OAK?

  • Apache Jackrabbit OAK
  • En JCR implementasjon (via utvidelser)
  • Lettvekts database for hierarkisk lagring
  • Enkel lettvekts datamodell
  • Effektiv lagring som GIT (immutable)
  • Støtter lange transaksjoner
  • Mange samtidige oppdateringer (MVCC)
  • Pluggbar validering, lagring og søk

  • Java Content Repository

  • Enkel datamodell (Noder og properties)
